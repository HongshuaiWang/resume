
## 简历（响应式web版）
> 网页版简历 

由于自身设计能力有限，该项目的项目源自[@ ToadWoo](http://resume.toadw.cn/) 的简历的启发

通过vue、elememt-ui、以及一些响应式技巧完成了此次的`响应式Web版简历`

需要自行修改的话，可以下载源代码，修改各个组建中的具体内容
- `work.vue` 组件为工作经历
- `project.vue` 组件为项目经历
- `skill.vue` 组件为专业技能
- `commit.vue` 组件为工作展望以及自我评价
- `myself.vue` 组件为基本信息
